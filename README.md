#

## 基于electron的图片浏览软件

图片文件存放于本地，目录信息存放在mysql中。

### 支持功能

搜索，添加图包，批量添加，收藏，文件夹，书签

#### 项目架构

#### `component` React组件文件夹

- `index` 主页面组件
- `detail` 图片详情页组件
- `style` 全局组件的样式
- `ImgComponent` 主页面中的图片组件
- `tsx文件` 全局组件

#### `config` 程序中的参数配置文件

#### `crawler` 集成的爬虫

#### `hooks` 自定义hooks

#### `icon` 图标

#### `ipc` electron文件

#### `server` 集成的服务器

#### `types` 定义的类型文件

#### `utils` 非视图的工具类
